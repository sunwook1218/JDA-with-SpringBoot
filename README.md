## Usage 

`ww command content`

1. ww s #userName : 사용자 검색
1-1. ww sd #userName : 사용자 검색 상세
2. ww i : 섬 일정 검색
3. ww r : 메세지 알림 Queue 등록
4. ww come : 채널 입장
5. ww quit : 채널 퇴장
6. ww system exit : 봇 강제 종료 명령어 (명령어 사용자 권한 필요)

0. DEBUG ww ~ : DEBUG 모드로 실행. error의 stackTrace를 메세지로 출력
